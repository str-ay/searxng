# Git Hooks

Этот каталог содержит git hooks для проекта.

## pre-commit

Блокирует коммиты с изменениями в submodule `mcp-searxng`.

Submodule предназначен только для чтения — все изменения должны идти через upstream репозиторий.

### Установка

```bash
git config core.hooksPath .githooks
```

Эта команда уже выполнена автоматически при клонировании (если настроено) или должна быть выполнена вручную.
